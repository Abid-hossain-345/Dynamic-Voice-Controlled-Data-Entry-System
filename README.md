## Description

A full-stack, voice-enabled data entry platform built with **Flask** and **MySQL**.  
This project allows dynamic creation and management of form fields, speech recognition input, PDF text extraction, and editable submissions. All data is stored in a MySQL database with auto-incremented IDs and flexible column management. Users can export the data to CSV for easy use.

---

## Features

- Dynamic form creation with add/remove columns  
- Speech recognition for hands-free data entry  
- Upload PDFs to auto-fill form fields  
- Editable submissions with real-time updates  
- Auto-increment ID tracking  
- CSV export of all submissions  
- Flexible database schema that updates automatically  
- Lightweight and responsive UI
